Create and reverse a list from 1 to an input integer

Hey there! This problem might be a bit unpredictable, but give it a go and let us know how you do!
You are given an integer 'x'. Your task is to create a MutableList of integers from 1 to 'x' and print it reversed. If 'x' is lesser than 1, print an empty list. You should get 'x' as input in a single line and print the reversed list as output in a single line with elements separated by a space.
Sample Input:
5

Sample Output:
5 4 3 2 1


Sample Input:
1

Sample Output:
1
